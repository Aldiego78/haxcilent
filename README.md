# haxcilent
A Crawler for hax/woiden run by GO 

-在使用前请遍历一次文件并填对所有参数否则无法运行

##必填参数
  
  两个网址
  
  telegrambot token
  
  chat id

##TO USE
-在使用前请遍历一次文件并填对所有参数否则无法运行
Debian/Ubuntu
```
apt install git golang -y && git clone https://github.com/IsMio/haxcilent.git && cd haxclient
```
```
go build main.go
```
运行
```
chmod +x main 
```
```
./main
```
运行后请使用screen或pm2之类的保活程序
